# HackNova-CodeVault

Hackathons often rely on GitHub links or manual checks to validate project submissions, making it difficult to prove whether a team submitted before the deadline or modified their project afterward. This leads to plagiarism concerns, disputes, and unfair evaluation.

The CodeVault is a secure submission platform designed to provide verifiable and tamper-proof hackathon project validation. Teams upload their final project (ZIP or Git repository), and the system generates a cryptographic hash and timestamp at the moment of submission. The hash, along with team and event details, is anchored on blockchain to create an immutable proof of submission. Any modification to files after submission changes the hash, instantly detecting tampering.

The platform also supports organizer-defined events, deadline enforcement, and automatic verification, removing reliance on trust-based manual reviews. Judges can independently verify authenticity, and participants gain proof of originality. Additional features include code similarity detection, public verification links, and QR-based certificates for winners.

Overall, the system improves fairness, transparency, and credibility of hackathons by ensuring projects are evaluated on provably unchanged submissions.
